How to make a game in 100+ hard steps

Hi everyone, over the past few weeks I have started making a 2D platformer/fighting roguelike with one of my friends. I am going to keep track of the project here. Starting off, the game engine we are using is Godot. Godot is a really cool game engine suitable for 2D and 3D game creation. I was first introduced to it by my friend when we made a couple game jam projects with it. I have tried using Godot and Unity, and I can safely say that I much prefer Godot. The node structure, and the language (GDScript) are great, and so far I have had a great time making the game.
